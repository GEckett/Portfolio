About Me

I'm an automotive project engineer with six years of experience in the industry, specializing in project management and process optimization. Certified in Internal Auditing to IATF 16949:2016, I hold a BSc in Motorsport Technology with a 2:1 grade from Oxford Brookes University.

At McLaren Automotive, I've played integral roles in the launch of iconic vehicle models like the McLaren 720S and 600LT. Currently serving as a Change Control Engineer, I implement change content to McLaren products through stakeholder management and process optimization, while also interpreting Power BI reports for data-driven decision-making.

Adept at working in collaborative environments, I've volunteered with the Oxford Universities Motorsport Foundation, gaining hands-on experience in car preparation for rally and circuit competition. I'm driven by continuous learning and am currently self-teaching Python coding alongside my full-time role.

With a keen eye for detail, strong organizational skills, and a proactive approach to problem-solving, I bring a dynamic skill set to every project I undertake.

Portfolio

Space Invaders

This script is a Python implementation of the classic arcade game "Space Invaders" using the Pygame library. The game faithfully replicates the gameplay mechanics and aesthetics of the original game, providing an engaging and nostalgic experience for players.

PDF to Audio

A Python script designed to facilitate the conversion of PDF documents into audiobook format. The script allows users to select a PDF file using a file explorer dialog, extract text from the PDF, convert the text to speech using Google Text-to-Speech (gTTS) API, and save the resulting audio as an MP3 file.

Image Watermarking

A Python script built using the Tkinter library for creating a graphical user interface (GUI) and the Pillow (PIL) library for image processing. This application allows users to easily add watermarks to images and save the watermarked images.

Online Shop

A full-stack web application built using the Flask framework in Python. It offers a modern and user-friendly online shopping experience, allowing users to browse through a wide range of products, add items to their cart, and securely complete their purchases.

F1 API Website

This Flask web application utilizes the Ergast API to fetch Formula 1 race information and results based on user input.Covers the entire duration of the championship to present day.

Typing Speed Test

A Python application designed to help users improve their typing speed and accuracy. The app features a user-friendly interface built using the Tkinter library, providing a seamless typing experience.
